# Инструкция по настройке Redmine для Content Agent

Руководство для администратора Redmine: создание custom fields и привязка к проекту.

---

## Полный список полей

### Обязательные поля (создать обязательно)

| Название поля в Redmine | Тип | Значения | Кто заполняет |
|------------------------|-----|----------|---------------|
| `Тип карточки` | Список (List) | hero, dimensions, colors, simple_benefit | Менеджер |
| `URL главного изображения` | Текст (Text) | HTTPS URL изображения | Менеджер |
| `Артикул` | Текст (Text) | Артикул товара, например SNK-001 | Менеджер |

### Дополнительные поля (создать по желанию)

| Название поля в Redmine | Тип | Описание | Кто заполняет |
|------------------------|-----|----------|---------------|
| `Вариант` | Текст (Text) | Вариант товара (цвет, размер и т.п.) | Менеджер |
| `Текст на карточке` | Текст (Text) | Основной текст/описание для карточки | Менеджер |
| `Цветовое решение` | Текст (Text) | Описание цветов (для типа `colors`) | Менеджер |
| `Сегмент инфографики` | Текст / Список | Сегмент инфографики (для типа `dimensions`) | Менеджер |
| `Маркетплейс` | Текст / Список | Целевой маркетплейс | Менеджер |
| `Номер изображения в карточке` | Текст (Text) | Номер слота изображения | Менеджер |
| `Формат изображения` | Текст / Список | png или jpg (по умолчанию png) | Менеджер |
| `Комментарий` | Текст (Text) | Заметки, не попадают в карточку | Менеджер |
| `Фотореференс` | Текст (Text) | URL фото-референса | Менеджер |

### Автоматические поля (создать, но не заполнять вручную)

| Название поля в Redmine | Тип | Описание | Кто заполняет |
|------------------------|-----|----------|---------------|
| `ID` | Текст (Text) | ID задания Celery | Пайплайн (автоматически) |
| `Рендер` | Текст (Text) | URL результирующего PNG | Пайплайн (автоматически) |

---

## Шаг 1. Войти как администратор

1. Войдите в Redmine под учётной записью с правами администратора.
2. В правом верхнем углу нажмите **Администрирование** (Administration).

---

## Шаг 2. Создать custom fields

Перейдите: **Администрирование → Пользовательские поля (Custom fields)**.

Нажмите **Новое пользовательское поле** и создайте поля по инструкции ниже.

## Частые ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Поле не отображается в задаче | Не привязано к проекту/трекеру | Проверьте «Использование» при редактировании поля |
| Pipeline: "card_type is required" | Поле `Тип карточки` не заполнено | Заполните поле в задаче |
| Pipeline: "main_image_url... required" | Поле `URL главного изображения` не заполнено | Укажите рабочий HTTPS URL |
| Пайплайн не видит поля | Название поля отличается от ожидаемого | Проверьте точное название в таблице выше |
